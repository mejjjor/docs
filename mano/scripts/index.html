Scripts

load.bat

Permet de charger la dernière version du site.

Il est conseillé d'executer ce script avant de commencer à travailer.

save.bat

Permet de sauvegarder les modifications effectuées et de publier le site en ligne.

Le login et mot de passe Github est requis pour mettre à jour le site. Comme on le sait, on oublie souvent de le faire .. :)

ATTENTION, il est nécessaire de terminer le script edit.bat avant d'executer save.bat.

edit.bat

Permet de visualiser les modifications du site en local

Lorsque vous lancez le script, une fenêtre s'ouvre permettant de générer le site en local. Fermez la fenêtre quand vous avez terminé vos modifications.

reset.bat

Efface les modifications depuis la dernière exécution de save.bat

Nécessaire quand on sait plus trop ce qu'on à fait .. :)
